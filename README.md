# Dribbble Lab

### 소개

이 프로젝트는 프론트엔드 개발자로서의 구현 방식을 표현하기 위해 제작한 Dribbble 기반 클론 프로젝트입니다.
실제 근무 중 작성한 코드를 공개할 수 없어, 클론 프로젝트를 제작하였습니다.

### 사용 기술

- TypeScript, React, NextJS, TailwindCSS, CVA, Zustand, ...
